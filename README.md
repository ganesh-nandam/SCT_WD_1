**SkillCraft Technology - Interactive Navigation Landing Page**

*A modern, responsive single-page landing website for SkillCraft Technology featuring an interactive fixed navigation bar with smooth scroll effects and beautiful hover animations.*

📋## **Project Overview**

**This is a single HTML file landing page with embedded CSS and JavaScript. It's lightweight, easy to deploy, and requires no external dependencies or build tools.**

**Live Features:**

**✨ Interactive fixed navigation bar, 
 🎨  Smooth color transitions on scroll,
 🖱️  Hover animations on menu items,
 📱  Fully responsive mobile design,
 ⚡  Smooth scrolling navigation,
 🎯  Active section tracking.**
